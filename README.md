# Site Cinema

Вебзастосунок кінотеатру на Django. Сервіс дає змогу переглядати афішу та деталі фільмів, обирати сеанси й місця в залі, бронювати квитки та керувати даними через адміністративну панель.

## Можливості

- каталог фільмів із постерами, жанрами й описами;
- розклад сеансів та інформація про кінозали;
- вибір вільних місць і створення бронювання;
- реєстрація, авторизація та профіль користувача;
- список власних бронювань;
- інтерфейс для працівників для перегляду бронювань і керування сеансами;
- Django admin для адміністрування контенту.

## Технології

- Python 3
- Django 5.2
- SQLite
- HTML/CSS

## Локальний запуск

1. Створіть та активуйте віртуальне середовище:

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

2. Встановіть залежності:

   ```powershell
   pip install -r requirements.txt
   ```

3. Застосуйте міграції:

   ```powershell
   python manage.py migrate
   ```

4. За потреби створіть демонстраційні фільми, зали та сеанси:

   ```powershell
   python manage.py seed_demo_data
   ```

5. Запустіть сервер розробки:

   ```powershell
   python manage.py runserver
   ```

Після запуску застосунок буде доступний за адресою `http://127.0.0.1:8000/`.

## Адміністрування

Щоб створити адміністратора, виконайте:

```powershell
python manage.py createsuperuser
```

Панель адміністратора доступна за адресою `http://127.0.0.1:8000/admin/`.

## Перевірка

```powershell
python manage.py test
```
